Class Array

The JavaScript Array Object. since FireFox 1.0, Netscape 2.0, and IE 4.0

Constructor

Array ( )

Properties

length - Object

Sets or returns the number of elements in an array. since FireFox 1.0, Netscape 2.0, and IE 4.0

prototype - Object

Allows you to add properties and methods to the object. since FireFox 1.0, Netscape 2.0, and IE 4.0

Methods

concat

Array concat ( )
The concat() method is used to join two or more arrays. This method does not change the existing arrays, it only returns a copy of the joined arrays. since FireFox 1.0, Netscape 4.0, and IE 4.0
Returns: Array
A new array containing the combined arrays.

join

String join ( separator )
The join() method is used to put all the elements of an array into a string. The elements will be separated by a specified separator. since FireFox 1.0, Netscape 3.0, and IE 4.0
Parameters:
separator <String> Optional. Specifies the separator to be used.
Returns: String
The array values as a string separeted by separator.

pop

Object pop ( )
The pop() method is used to remove and return the last element of an array. This method changes the length of the array. since FireFox 1.0, Netscape 4.0, and IE 5.5
Returns: Object
The last element in the array.

push

Array push ( newelement1 , newelementX )
The push() method adds one or more elements to the end of an array and returns the new length. This method changes the length of the array. since FireFox 1.0, Netscape 4.0, and IE 5.5
Parameters:
newelement1 <Object> Required. The first object to append.
newelementX <Object> Optional. Any number of additional objects parameters to append.
Returns: Array
The array calling method with new elements.

reverse

Array reverse ( )
The reverse() method is used to reverse the order of the elements in an array. The reverse() method changes the original array. since FireFox 1.0, Netscape 3.0, and IE 4.0
Returns: Array
The array calling method.

shift

Object shift ( )
The shift() method is used to remove and return the first element of an array. This method changes the length of the array. since FireFox 1.0, Netscape 4.0, and IE 5.5
Returns: Object
The first element in the array.

slice

Array slice ( start , end )
The slice() method returns selected elements from an existing array. You can use negative numbers to select from the end of the array. If end is not specified, slice() selects all elements from the specified start position and to the end of the array. since FireFox 1.0, Netscape 4.0, and IE 4.0
Parameters:
start <Number> Required. Specify where to start the selection. Must be a number.
end <Number> Optional. Specify where to end the selection. Must be a number.
Returns: Array
The elements removed from the array.

sort

Array sort ( sortby )
The sort() method is used to sort the elements of an array. The sort() method will sort the elements alphabetically by default. However, this means that numbers will not be sorted correctly (40 comes before 5). To sort numbers, you must create a function that compare numbers. After using the sort() method, the array is changed. since FireFox 1.0, Netscape 3.0, and IE 4.0 sortby for number sorting is: function sortNumber(a,b){return a - b;}
Parameters:
sortby <Function> Optional. Specifies the sort order. Must be a function.
Returns: Array
The array calling method.

splice

Array splice ( index , howmany , element1 , elementX )
The splice() method is used to remove and add new elements to an array. since FireFox 1.0, Netscape 4.0, and IE 5.5
Parameters:
index <Number> Required. Specify where to add/remove elements. Must be a number.
howmany <Number> Required Specify how many elements should be removed. Must be a number, but can be "0".
element1 <Object> Optional. Specify a new object to add to the array.
elementX <Object> Optional. Any number of additional objects parameters to append.
Returns: Array
The array calling method with new elements.

toSource

String toSource ( )
Returns the source code used to generate array. This method does not work in Internet Explorer! since FireFox 1.0 and Netscape 4.0
Returns: String
The source-code that generated the array.

toString

String toString ( )
The toString() method converts a array value to a String and returns the result. The elements in the array will be separated with commas. since FireFox 1.0, Netscape 3.0, and IE 4.0
Returns: String
The array as a String.

unshift

Number unshift ( )
The unshift() method adds one or more elements to the beginning of an array and returns the new length. This method changes the length of the array. The unshift() method does not work properly in Internet Explorer! since FireFox 1.0, Netscape 4.0, and IE 6.0
Returns: Number
The new length of the array.

valueOf

Array valueOf ( )
The valueOf() method returns the primitive value of an Array object. The primitive value is inherited by all objects descended from the Array object. The valueOf() method is usually called automatically by JavaScript behind the scenes and not explicitly in code. since FireFox 1.0, Netscape 2.0, and IE 4.0
Returns: Array
The array primitive.
